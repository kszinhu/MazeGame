keyboard | Movimentação

Utilize as setas do teclado para mover em sua respectiva direção.

arrow_upward - Para rotacionar a visão do personagem para Cima

arrow_downward - Para rotacionar a visão do personagem para Baixo

arrow_forward - Para mover o personagem à Direita

arrow_back - Para mover o personagem à Esquerda

space_bar - Matenha pressionado para Contrair o personagem
